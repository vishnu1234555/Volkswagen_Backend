# Volkswagen RAG Backend

This is the backend API for the VW RAG chatbot. It runs a single-file Flask server connecting to Groq (LLM) and Qdrant (Vector Database).

## Prerequisites
* Python 3.x
* Qdrant instance running locally or via cloud
* Groq API Key
